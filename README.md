# Техническое задание  
## Tetris Vashchenko Secret: Victoria Edition  

**Проект:** Модернизированный тетрис  

---

## 1. Цель проекта  
Разработать веб-сайт с игрой в тетрис для студентов группы **ФТ-202-1** с возможностью:  

- играть в тетрис в браузере  
- сохранять результаты игроков  
- отображать таблицу лидеров  
- наблюдать прекрасность Вани Ващенко 😎  
- дуэли 1 vs 1 со зрительным залом
- реакции  

---

## 2. Описание проекта  
Проект представляет собой веб-приложение, содержащее реализацию игры «Тетрис» с дополнительными функциями.  

Пользователь заходит на сайт, вводит своё имя, играет в тетрис, после окончания игры результат сохраняется в базе данных и отображается в таблице лучших игроков.  
Сайт должен поддерживать несколько пользователей и хранить статистику игр.  

Также на сайте должен присутствовать отдельный блок, посвящённый **Ване Ващенко**, содержащий изображение, текст или анимацию, демонстрирующую его прекрасность ✨  

---

## 3. Требования к интерфейсу  

### Главная страница должна содержать:  
- кнопку **Играть**  
- кнопку **Топ игроков**  
- кнопку **О проекте / Ваня Ващенко**  

### Страница игры должна содержать:  
- игровое поле тетриса  
- отображение текущего счёта  
- отображение следующей фигуры  
- кнопку перезапуска  

### Страница топа игроков должна содержать:  
- таблицу результатов  
  - имя игрока  
  - количество очков  
  - дату игры  

### Страница Вани Ващенко должна содержать:  
- изображение  
- текстовое описание  

**Интерфейс должен быть:**  
- понятным  
- удобным  
- адаптированным под ПК  

---

## 4. Функциональные требования  

- Реализация классического тетриса  
- Подсчёт очков  
- Ускорение игры со временем  
- Сохранение результатов в базу данных  
- Вывод топ‑10 игроков  
- Возможность играть несколько раз
