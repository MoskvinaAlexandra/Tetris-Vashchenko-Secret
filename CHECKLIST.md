## 🔧 Техническая реализация

### Backend (Node.js + Express)
- [x] **Express server** (port 3000)
- [x] **WebSocket (ws)** для 1v1 дуэлей
- [x] **PostgreSQL** для хранения очков
- [x] **REST API** для топа и сохранения

### Frontend (Vanilla JS)
- [x] **HTML5** семантика
- [x] **Canvas 2D** для рендеринга
- [x] **WebSocket client** для синхронизации
- [x] **CSS3** с Neon Green темой

### Структура
- [x] `client/` — публичная часть (не трогаем командиков)
- [x] `server/` — серверная логика (можно менять)
- [x] Разумная организация файлов
