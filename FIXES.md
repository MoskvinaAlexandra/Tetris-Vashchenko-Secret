# Tetris Fix Summary 🎮

## 🔧 Исправленные проблемы

### 1. ❌ → ✅ Кнопка "Создать комнату" не работала

**Проблема:** Синтаксическая ошибка в `client/js/game.js` — функция `handleMessage()` была задублирована и содержала логические ошибки.

**Решение:** 
- Переписал `handleMessage()` с использованием `switch` вместо вложенных `if-else`
- Удалил дублированный код
- Теперь кнопка работает корректно → открывает лобби → показывает код комнаты

### 2. ❌ → ✅ WebSocket ошибки на сервере

**Проблема:** 
- импорт `WebSocket` не был добавлен
- Использовалась ссылка на переменную `gameStarted` (не определена)
- Проверки `readyState` были неправильными

**Решение:**
- Добавил `import { WebSocket }` в `server/server.js`
- Исправил все ссылки `client.OPEN` → `WebSocket.OPEN`
- Удалил ветвь кода с несуществующей переменной `gameStarted`
- Добавил `process.env.PORT` для гибкого выбора порта

### 3. ❌ → ✅ Движение фигур не работало

**Проблема:** 
- `sendMove()` просто отправлял сообщение, не применяя локальное движение
- `gameLoop()` ссылался на неопределённую переменную `roomHasOpponent`
- Нет логики обработки arrow keys

**Решение:**
- Реализовал полную логику в `sendMove()`:
  - Применяется движение локально
  - Отправляется состояние доски по WebSocket
  - Поддержаны: left/right/down/rotate/drop
- Упростил `gameLoop()` для корректного рендеринга обеих досок

### 4. ❌ → ✅ БД инициализация сложной была

**Проблем:** Нет скрипта для создания таблицы `scores`

**Решение:**
- Создал `server/init-db.js` для автоматического создания таблицы
- Добавил script `npm run init-db` в package.json
- Таблица создаётся с индексом по score для быстрого получения top-10
